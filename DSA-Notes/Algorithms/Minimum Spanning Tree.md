# Minimum Spanning Tree

## 📌 Definition
A spanning tree of a connected, weighted graph with the minimum possible total edge weight. A spanning tree connects all V vertices with exactly V-1 edges and no cycles.

## 🧠 Algorithms
| Algorithm | Time | Best For |
|-----------|------|---------|
| [[Kruskal's Algorithm]] | O(E log E) | Sparse graphs |
| [[Prim's Algorithm]] | O(E log V) | Dense graphs |

## 🔥 Key Properties
- Exactly V-1 edges
- No cycles
- Connects all vertices
- May not be unique (if edge weights are equal)
- Cut property: minimum weight edge crossing any cut belongs to some MST
