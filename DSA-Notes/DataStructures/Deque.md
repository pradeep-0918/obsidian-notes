---

## 📌 Definition
A **deque** (double-ended queue) allows insertions and deletions at **both ends** in O(1).

---

## ⚙️ How It Works
```
← front [1][2][3][4] back →
push_front / pop_front | push_back / pop_back
```

---

## 🧠 When to Use
- Sliding window maximum/minimum
- BFS with bidirectional expansion
- Monotonic queue problems
- Palindrome checking

---

## ⏱ Complexity

| Operation | Time |
|---|---|
| Push front/back | O(1) |
| Pop front/back | O(1) |
| Peek | O(1) |
