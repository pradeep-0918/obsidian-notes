# 📦 Hash Set

#DataStructure #HashBased

⬅️ [[Hash Map]] | ➡️ [[Bloom Filter]]

---

## 📌 Definition

A **Hash Set** stores **unique keys** (no values). Provides O(1) membership testing.

---

## ⚙️ Usage

```python
s = set()
s.add(x)
s.remove(x)
x in s       # O(1)
s1 & s2      # intersection
s1 | s2      # union
s1 - s2      # difference
```

---

## 🧠 When to Use

- Check for duplicates
- Track visited nodes in BFS/DFS
- Set operations
