---

## 📌 Definition

A **Hash Map** maps **keys to values** using a hash function. Provides O(1) average-case insert, delete, lookup.

---

## ⚙️ Usage

```python
# Python dict
d = {}
d['key'] = 'value'
val = d.get('key', default)
del d['key']
'key' in d  # O(1) membership
```

---

## 🧠 When to Use

- Two Sum (complement lookup)
- Frequency counting
- Memoization / caching
- Grouping (anagrams)
