## 📌 Definition
An **octree** is a tree where each internal node has **8 children**, used to partition 3D space by recursively subdividing into 8 octants.

---

## ⚙️ How It Works
Each 3D region divides into 8 equal sub-regions:
- Front/Back × Left/Right × Top/Bottom

---

## 🧠 When to Use
- 3D spatial indexing
- 3D game engines (visible set computation)
- 3D collision detection
- Point cloud processing
