## 📌 Definition

A **Circular Linked List** is a linked list where the last node points back to the **head**, forming a circle.

```
[1|→] → [2|→] → [3|→] ↩ (back to [1])
```

---

## 🧠 When to Use

- Round-robin scheduling
- Music playlist loops
- Buffer implementations
