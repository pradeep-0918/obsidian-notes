## 📊 Generation Number System

> 💡 **The most powerful shortcut in Blood Relations!**

Assign generation numbers to every person:

| Generation | Who |
|-----------|-----|
| **+3** | Great-Grandparents |
| **+2** | Grandparents |
| **+1** | Parents, Uncles, Aunts |
| **0** | YOU, Brothers, Sisters, Cousins (same age) |
| **−1** | Sons, Daughters, Nieces, Nephews |
| **−2** | Grandsons, Granddaughters |
| **−3** | Great-Grandchildren |

### How to Use Generation Numbers:

When you go:
- **UP** the tree (towards grandparents) → **Add generations (+)**
- **DOWN** the tree (towards children) → **Subtract generations (−)**

## 🧠 The "Who Introduced Them" Trick

Whenever you see "introduced by", that person is the **bridge**.

**Q: "A introduced B to C saying, B is the son of A's father's only daughter."**

Break it:
- A's father's only daughter → A's **Sister** (since father has only one daughter)
- Sister's son → A's **Nephew**

✅ B is A's **Nephew**

## 🎯 Special Cases to Memorize

### Case 1: "Only Son / Only Daughter"
> If someone says "only son" or "only daughter" → **it means that person = the speaker's parent or sibling**

**Example:**
- "My grandfather's only son" = My **Father** (if grandfather has only one son)
- "My mother's only daughter" = **Me** (or my sister, if I'm male)

---

### Case 2: "Pointing to a photograph"
> These questions say "Pointing to a photo, X says..."
> Start from X, build the tree forward.

---

### Case 3: Who is sitting / standing / pointing?
> These are **puzzle-type** questions. Draw immediately. See [[03_Types_of_Questions]] for full breakdown.
