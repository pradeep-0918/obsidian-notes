## 🚀 The "SCAN → DECODE → DRAW → ANSWER" Framework

Before reading any shortcut, burn this 4-step into your brain:

```
📖 SCAN    → Read the question fully once (5 seconds)
🔓 DECODE  → Identify question TYPE (2 seconds)
✏️ DRAW    → Build the mini family tree (10 seconds)
✅ ANSWER  → Name the relation with correct gender (3 seconds)
```

Total: **~20 seconds per question** (reduces to 10s with practice)

---

## ⚡ SHORTCUT #1 — The "Only" Magic Word

> Whenever you see **"only son"** or **"only daughter"** — it is pointing directly at one specific person.

| Phrase | Who It Means |
|--------|-------------|
| "My grandfather's only son" | = My **Father** (direct) |
| "My mother's only daughter" | = **Me** (if I'm female) or **My sister** |
| "His father's only daughter" | = His **Sister** |
| "Her mother's only son" | = Her **Brother** |

**Formula:**
> Person's Parent's **Only** [gender] → That [gender] sibling or the person themselves

---

## ⚡ SHORTCUT #2 — The Mirror Trick

> Every relation has a **reverse relation**. When confused, flip the path.

```
If A is the FATHER of B
→ B is the SON/DAUGHTER of A

If A is the UNCLE of B
→ B is the NEPHEW/NIECE of A

If A is the GRANDFATHER of B
→ B is the GRANDSON/GRANDDAUGHTER of A
```

**Use Case:**
> Q: "How is Ravi related to Meena?" gives too many steps?
> Instead: "How is Meena related to Ravi?" (if that's shorter) → then **flip the answer**

---

## ⚡ SHORTCUT #3 — Generation Jump Formula

> Count the **generation gap** → instantly know the prefix (grand/great-grand)

| Generation Gap | Relation Name |
|---------------|---------------|
| ±1 | Father/Son, Mother/Daughter |
| ±2 | Grandfather/Grandson |
| ±3 | Great-grandfather/Great-grandson |
| 0 | Brother/Sister/Cousin |

**Example:**
> "A's great-grandfather's son's son's son = ?"
> Jump: +3 (up) then −1 −1 −1 (down) = net jump: 0 = **same generation = sibling/cousin**

---

## ⚡ SHORTCUT #4 — The Couple Detection Trick

> In puzzle questions, find couples **first**. Everything else falls into place.

**Clue words that reveal couples:**
- "wife of" / "husband of"
- "married to"
- "daughter-in-law" (points to son + wife = couple above)
- "son-in-law" (points to daughter + husband = couple above)

**Rule:**
> Number of couples = Number of generations − 1 (usually)

---

## ⚡ SHORTCUT #5 — The "Nephew/Niece" Shortcut

> If the question involves brothers/sisters and their children → it's **nephew or niece**.

```
Brother's child = Nephew (M) / Niece (F)
Sister's child  = Nephew (M) / Niece (F)
```

> Exam Trap: "A's brother's daughter is A's ___?"  
> Answer: **Niece** (not daughter, not cousin!)

---

## ⚡ SHORTCUT #6 — The "Cousin Never Lies" Rule

> Uncle or Aunt's child is ALWAYS a **cousin** — no matter how many uncles or aunts.

```
Father's Brother's Child    = Cousin
Father's Sister's Child     = Cousin
Mother's Brother's Child    = Cousin
Mother's Sister's Child     = Cousin
```

---

## ⚡ SHORTCUT #7 — The In-Law Web

> All "in-law" relations come from **marriage**. One rule covers all:

```
Spouse's [relation] = Your [relation]-in-law
```

| Path | Result |
|------|--------|
| Husband's Father | Father-in-law |
| Wife's Mother | Mother-in-law |
| Husband's Brother | Brother-in-law |
| Wife's Sister | Sister-in-law |
| Son's Wife | Daughter-in-law |
| Daughter's Husband | Son-in-law |

---

## ⚡ SHORTCUT #8 — The "Cannot Be Determined" Trap

> Some questions don't give gender → Answer is "**Cannot be determined**"

**When to use it:**
- Person could be male OR female (not specified)
- Person could be a son OR daughter (not specified)

**Example:**
> "A is the child of B. B is the sibling of C. How is C related to A?"
> C could be Uncle (if C is male) OR Aunt (if C is female) → **Cannot be determined**

---

## 🧠 Bonus: The "Step-Count" Mental Model

For super long chain questions, use **step counting**:

```
Each step UP (+1) = going toward grandparent
Each step DOWN (−1) = going toward child
Each step SIDEWAYS (0) = sibling/cousin level
```

After counting steps:
- Net +2 or above → "Grand" prefix
- Net 0 → Same generation (sibling/cousin)
- Net -2 or below → "Grand" suffix

## 🏋️ Practice Drill — Apply Shortcuts

Try these without drawing (speed mode):

1. "My father's only sister's son" = ? *(Cousin)*
2. "My grandfather's son's daughter" = ? *(Sister/Cousin)*
3. "My sister's husband's father" = ? *(Father-in-law's father — no standard name)*
4. "My mother's brother's only daughter" = ? *(Maternal cousin — Female)*
5. "My son's wife's mother" = ? *(Daughter-in-law's mother — no standard name)*

Answers: 1-Cousin, 2-Sister or Cousin (depends), 3-N/A, 4-Cousin(F), 5-N/A
