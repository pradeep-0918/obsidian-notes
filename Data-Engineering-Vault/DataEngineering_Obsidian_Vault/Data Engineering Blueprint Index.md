## 🚀 Learning Roadmap

### Phase 1: Foundation (Months 1–2)
```
Week 1–2:  SQL (Advanced) → Window functions, CTEs, Optimization
Week 3–4:  Python for Data → Pandas, PySpark basics
Week 5–6:  Projects 1, 2, 3 → CSV, Scraping, POS ETL
Week 7–8:  Cloud Basics → AWS S3, EC2, IAM fundamentals
```

### Phase 2: Intermediate (Months 3–4)
```
Week 9–10:  Apache Spark (Deep Dive) → RDDs, DataFrames, Spark SQL
Week 11–12: Apache Kafka → Topics, Partitions, Consumers/Producers
Week 13–14: Projects 4–8 → Logs, Clickstream, IoT
Week 15–16: Airflow → DAGs, Operators, Scheduling
```

### Phase 3: Advanced (Months 5–6)
```
Week 17–18: Streaming Architectures → Lambda, Kappa, Delta
Week 19–20: Cloud Deep Dive → Redshift, BigQuery, Synapse
Week 21–22: Projects 11–18 → Real-time, FAANG-level systems
Week 23–24: System Design → Design a data platform end-to-end
```

### Phase 4: Job-Ready (Month 7+)
```
Interview prep, Portfolio building, GitHub optimization, LinkedIn strategy
→ See [[Data Engineer Career Roadmap]] and [[Top 1% Data Engineer Strategy]]
```
