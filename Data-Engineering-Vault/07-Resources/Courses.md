## 🗺️ Structured Learning Path

### Month 1-2: Foundations
```
1. SQL (2 weeks)
   → Mode Analytics SQL Tutorial (free)
   → Practice on LeetCode (Easy SQL)
   
2. Python (2 weeks)
   → Python for Everybody - Coursera (free audit)
   → Automate the Boring Stuff with Python (free online)

3. Linux/Bash (3 days)
   → The Linux Command Line (free PDF)
   → Practice in WSL/Ubuntu terminal
```

### Month 3-4: Core Tools
```
4. Databases (2 weeks)
   → PostgreSQL Tutorial (postgresqltutorial.com)
   → Build: Simple CRUD app with psycopg2

5. Cloud Basics (2 weeks)
   → AWS Cloud Practitioner Essentials (free, AWS)
   → Build: S3 bucket with Python SDK

6. First Pipeline (2 weeks)
   → Build weather pipeline (see [[05-Projects/Beginner-Projects]])
```

### Month 5-6: Distributed Systems
```
7. Apache Spark (4 weeks)
   → Databricks Academy: Apache Spark (free)
   → Build: Spark job on EMR

8. Apache Kafka (2 weeks)
   → Confluent Developer: Kafka for Beginners (free)
   → Build: Producer/Consumer pipeline
```

### Month 7-8: Production Skills
```
9. Airflow (2 weeks)
   → Astronomer Academy (free)
   → Build: Multi-task DAG with sensors

10. dbt (2 weeks)
    → dbt Learn Fundamentals (free, official)
    → Build: dbt project with 10+ models

11. Cloud Data Warehouse (2 weeks)
    → Snowflake: Getting Started (free)
    → or BigQuery Qwiklabs
```

## 🏅 Free Quality Courses

### DataTalks.Club Data Engineering Zoomcamp
**Cost:** Free | **Duration:** 9 weeks | **Format:** Live + recorded

The single best free DE course available. Covers:
- Containerization (Docker, Kubernetes)
- Workflow orchestration (Airflow/Prefect/Kestra)
- Data warehouse (BigQuery)
- Batch processing (Spark)
- Streaming (Kafka)
- Capstone project

**URL:** datatalks.club/courses/data-engineering-zoomcamp

### dbt Fundamentals
**Cost:** Free | **Duration:** 5 hours
Official course from dbt Labs. Essential.
**URL:** courses.getdbt.com

### Databricks Academy - Apache Spark
**Cost:** Free tier | **Duration:** 20 hours
Official Spark training from the creators.
**URL:** academy.databricks.com

### AWS Skill Builder - Data Analytics
**Cost:** Free | **Duration:** Varies
Official AWS training for all data services.
**URL:** skillbuilder.aws
