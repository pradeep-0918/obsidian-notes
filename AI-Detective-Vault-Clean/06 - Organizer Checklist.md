# 🧰 Organizer Checklist

> 🔗 Back to [[00 - Home]]  
> 📎 Full timeline → [[05 - Run Sheet]]

---

## 📅 1 Week Before

- [ ] Confirm venue and projector/screen availability
- [ ] Finalize number of participants (ideal: 6–10)
- [ ] Print all [[03 - Case Files]] on separate cards
- [ ] Confirm 3 judges (2 faculty + 1 guest/senior)
- [ ] Share [[04 - Judging and Evaluation]] criteria with judges in advance
- [ ] Print [[07 - Score Sheet]] — one per judge
- [ ] Arrange prize and certificates
- [ ] Create a registration form (Google Form)
- [ ] Prepare event poster with name: **"AI Prompting Engineer — AI Detective"**

---

## 📅 1 Day Before

- [ ] Test projector and screen display
- [ ] Test AI tools on the venue computer (ChatGPT/Claude accessible?)
- [ ] Confirm all participants are registered
- [ ] Prepare participant name cards or badges
- [ ] Prepare a countdown timer (online or app)
- [ ] Brief co-organizer on their role during the event
- [ ] Prepare background music playlist (15 min)
- [ ] Print extra score sheets as backup

---

## 📅 Day of Event

- [ ] Arrive 30 minutes early
- [ ] Set up projector with demo slides ready
- [ ] Place case file cards face-down on tables
- [ ] Set up judge table with printed score sheets and pens
- [ ] Test AI tool is working and accessible from participant seats
- [ ] Confirm prize and certificates are on hand
- [ ] Brief judges one more time on scoring

---

## 🎤 Host Script Outline

1. *"Welcome everyone to AI Prompting Engineer!"*
2. *"Today's challenge is called AI Detective…"* (explain concept in 2 min)
3. Show bad prompt vs good prompt on screen (live demo)
4. *"Here are the rules…"* (briefly go through [[02 - Rules and Format]])
5. *"You have 15 minutes. Your time starts… now!"*
6. During presentations: *"Please welcome [Name] to the stage!"*
7. Final: *"After careful deliberation, today's AI Detective is…"*
