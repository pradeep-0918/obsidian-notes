## 🚀 TRICK 1 — The Age Difference Lock

> **"Age difference NEVER changes with time."**

### How to use it:
If A is older than B by **D years**, then:
- In any past or future → A is still D years older

### Speed Application:
> *"5 years ago, A was 30. B is 8 years younger than A. What is B's age now?"*

Normal way: 3 steps
**Shortcut:**
- A now = 30 + 5 = 35
- B now = 35 − 8 = **27** ← done in 2 seconds!

## 🚀 TRICK 2 — The Multiple Flip Trick

> When someone is "n times older", build ratio and flip.

### Pattern:
*"A is 3 times B now. After x years, A is 2 times B."*

**Shortcut Formula:**
```
A's future age / B's future age = new ratio

(Present A + x) / (Present B + x) = new multiple
```

Use: Cross multiply → solve for one variable.

### 5-Second Version:
Set up: Let B = y → A = 3y
→ (3y + x) / (y + x) = 2
→ 3y + x = 2y + 2x
→ **y = x**

**So B's present age = the number of years added!** (when ratio changes from 3:1 to 2:1)

## 🚀 TRICK 3 — Sum + Difference = Instant Answer

> If sum and difference of two ages are given:

```
Larger age  = (Sum + Difference) ÷ 2
Smaller age = (Sum − Difference) ÷ 2
```

### Example:
Sum of A and B = 40, Difference = 10

- A (larger) = (40 + 10) / 2 = **25**
- B (smaller) = (40 − 10) / 2 = **15**

⏱️ No algebra needed. Pure arithmetic. **2 seconds!**

## 🚀 TRICK 4 — The Ratio Shortcut (When Sum is Given)

> Ratio a:b and Sum = S

```
Larger  = S × b/(a+b)     [if b > a]
Smaller = S × a/(a+b)
```

### Example:
Ratio 2:3, Sum = 50

- Smaller = 50 × 2/5 = **20**
- Larger  = 50 × 3/5 = **30**

**Zero algebra. Just multiply fractions!**

## 🚀 TRICK 5 — The "After How Many Years" Trick

> *"Currently A:B = m:n. After how many years will ratio be p:q?"*

```
Let years = t
(A + t)/(B + t) = p/q

Cross multiply → solve t
```

**Ultra shortcut (when one ratio term = 1):**
If ratio becomes 1:1 (equal ages), that's impossible unless ages are equal!
If asked "when will A be twice B" → use t = (mB − nA... but just cross multiply, it's fast)

## 🚀 TRICK 6 — Average Age Trick

> If average age of a group = A, and total people = n:
> **Total sum of ages = A × n**

When a new person joins:
```
New sum = Old sum + New person's age
New average = New sum / (n + 1)
```

**Reverse trick:** If average increases by d when someone leaves:
```
Leaving person's age = Old average − d × (n−1)
```

## 🚀 TRICK 7 — The "X Years Ago" Equation Hack

> "n years ago, A was k times B"

**One-line setup:**
```
(A − n) = k × (B − n)
```
→ A − n = kB − kn
→ A = kB − kn + n
→ **A = kB − n(k−1)**

Memorize: **A = kB − n(k−1)**

### Example:
10 years ago, father was 4 times son. Son now = 15.
→ Father = 4(15) − 10(4−1) = 60 − 30 = **30**

⏱️ One substitution. **3 seconds!**

---

## 🚀 TRICK 8 — The "Hence / After" Equation Hack

> "n years later, A will be k times B"

**One-line setup:**
```
(A + n) = k × (B + n)
→ A = kB + n(k−1)
```

Memorize: **A = kB + n(k−1)**
