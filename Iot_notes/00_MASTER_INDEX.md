## 🧠 Quick Revision — Key Definitions

| Term | One-Line Definition |
|------|-------------------|
| **IoT** | Network of physical devices connected to the internet, sharing data automatically |
| **Arduino** | Open-source microcontroller platform with setup() and loop() structure |
| **Raspberry Pi** | Credit-card Linux computer with GPIO for IoT hardware control |
| **MQTT** | Lightweight Publish-Subscribe messaging protocol for IoT |
| **ZigBee** | Low-power mesh wireless protocol (IEEE 802.15.4) for IoT |
| **Smart Sensor** | Sensor + Processor + Communication combined in one |
| **GPIO** | General Purpose Input/Output pins for hardware interfacing |
| **PWM** | Pulse Width Modulation — simulates analog output using digital signal |
| **API** | Rules/protocols enabling software systems to communicate |
| **Edge Computing** | Processing data locally (near device) instead of sending to cloud |
