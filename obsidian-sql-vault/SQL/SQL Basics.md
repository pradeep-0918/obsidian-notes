# SQL Basics — Map of Content

> [!abstract] Overview SQL (Structured Query Language) is divided into **4 sublanguages**, each with a distinct role in managing databases.

## The 4 Sublanguages

```
SQL
├── DDL — Data Definition Language   → Structure
├── DML — Data Manipulation Language → Data
├── TCL — Transaction Control Language → Safety
└── DCL — Data Control Language      → Access
```

---

## Quick Reference

|Sublanguage|Full Name|Commands|Auto-Commit|
|---|---|---|---|
|[[DDL - Data Definition Language\|DDL]]|Data Definition Language|CREATE, ALTER, DROP, TRUNCATE, RENAME|✅ Yes|
|[[DML - Data Manipulation Language\|DML]]|Data Manipulation Language|SELECT, INSERT, UPDATE, DELETE, MERGE|❌ No|
|[[TCL - Transaction Control Language\|TCL]]|Transaction Control Language|COMMIT, ROLLBACK, SAVEPOINT|❌ No|
|[[DCL - Data Control Language\|DCL]]|Data Control Language|GRANT, REVOKE|✅ Yes|
