## 🧠 SECTION 1: MULTIPLICATION TRICKS

### 1.1 Multiplying by Powers of 2
```
×2:  Double the number
×4:  Double twice
×8:  Double three times
×16: Double four times

Example: 47 × 8 = 47×2×2×2 = 94×2×2 = 188×2 = 376
Example: 35 × 16 = 35×2×2×2×2 = 70×2×2×2 = 140×2×2 = 280×2 = 560
```

### 1.2 Multiplying by 5, 25, 125
```
×5:   n×10÷2         (47×5 = 470÷2 = 235)
×15:  n×10 + n×5     (47×15 = 470+235 = 705)
×25:  n×100÷4        (47×25 = 4700÷4 = 1175)
×75:  n×100×3÷4      (47×75 = 4700×3÷4 = 14100÷4 = 3525)
×125: n×1000÷8       (24×125 = 24000÷8 = 3000)
×250: n×1000÷4       (36×250 = 36000÷4 = 9000)
×500: n×1000÷2       (47×500 = 47000÷2 = 23500)
```

### 1.3 Multiplying by 9, 11, 12, 15, 99
```
×9:   n×10 − n        (47×9 = 470−47 = 423)
×11:  n×10 + n        (47×11 = 470+47 = 517)
×12:  n×10 + n×2      (47×12 = 470+94 = 564)
×15:  n×10 + n×5      (47×15 = 470+235 = 705)
×19:  n×20 − n        (47×19 = 940−47 = 893)
×21:  n×20 + n        (47×21 = 940+47 = 987)
×99:  n×100 − n       (47×99 = 4700−47 = 4653)
×101: n×100 + n       (47×101 = 4700+47 = 4747)
```

### 1.4 Multiplying Two Numbers Near Same Round Number
```
"Hub and Spoke" method:

Both near 50:
  48 × 52 = (50−2)(50+2) = 2500−4 = 2496
  47 × 53 = (50−3)(50+3) = 2500−9 = 2491
  45 × 55 = (50−5)(50+5) = 2500−25 = 2475

Both near 100:
  96 × 104 = (100−4)(100+4) = 10000−16 = 9984
  97 × 103 = 10000−9 = 9991
  
Both near 200:
  198 × 202 = (200−2)(200+2) = 40000−4 = 39996
  
Both near 30:
  28 × 32 = (30−2)(30+2) = 900−4 = 896
  29 × 31 = 900−1 = 899
```

### 1.5 The "Split and Multiply" Method
```
For any larger multiplication, split one number:

34 × 56:
  = 34 × 50 + 34 × 6
  = 1700 + 204
  = 1904 ✓

67 × 84:
  = 67 × 80 + 67 × 4
  = 5360 + 268
  = 5628 ✓

123 × 47:
  = 100×47 + 23×47
  = 4700 + 1081
  = 5781 ✓
```

### 1.6 Cross-Multiplication (2×2 digits, mental)
```
The box method — left to right:
  Tens × Tens  |  Tens × Units (both) | Units × Units
  
  47 × 83:
  4×8=32 | 4×3+7×8=68 | 7×3=21
  32 | 68 | 21
  32+6=38 | 8+2=10 | 1 [carry rightward-leftward]
  
  Wait: read right to left with carries:
  units: 1 (from 21, carry 2)
  tens:  68+2=70→0, carry 7
  hundreds: 32+7=39
  → 3901 ✓
```

## 🧠 SECTION 2: DIVISION TRICKS

### 2.1 Dividing by 5, 25, 125
```
÷5:   ×2 then ÷10       (235÷5 = 470÷10 = 47)
÷25:  ×4 then ÷100      (175÷25 = 700÷100 = 7)
÷50:  ×2 then ÷100      (350÷50 = 700÷100 = 7)
÷125: ×8 then ÷1000     (875÷125 = 7000÷1000 = 7)
÷250: ×4 then ÷1000     (750÷250 = 3000÷1000 = 3)
```

### 2.2 Dividing by 9, 11
```
For simple cases:
÷9:  Sum of digit groups (99 rule):
  837÷9: digit sum = 18 → divisible; 837÷9=93
  
  Quick check: If digit sum divisible by 9, then ÷9 is exact
  Otherwise: remainder = digit sum mod 9
  
÷11: Alternating sum check
  1342÷11: 1−3+4−2=0 → exactly divisible
  1342÷11=122
```

### 2.3 Divide Large Numbers by Breaking Down
```
1248 ÷ 12:
  = (1200 + 48) ÷ 12
  = 100 + 4
  = 104 ✓

5765 ÷ 25:
  = 5765 × 4 ÷ 100
  = 23060 ÷ 100
  = 230.6 ✓
```

## 🧠 SECTION 3: PERCENTAGE TRICKS

### 3.1 The Core 10-1 Method
```
Any % = combination of 10% chunks and adjustments

10% of N: move decimal left one place
  10% of 340 = 34
  10% of 1250 = 125

1% of N: move decimal left two places
  1% of 340 = 3.4

Build any %:
  15% = 10% + 5%
  25% = 10%×2 + 5%
  35% = 10%×3 + 5%
  8%  = 10% − 2%  (or: 5% + 3% = 5% + 1%×3)
  17% = 10% + 7%  (or: 10% + 5% + 2%)
```

### 3.2 Key % Shortcuts
```
20% of N = N/5       (divide by 5)
25% of N = N/4       (divide by 4)
33.33% of N = N/3    (divide by 3)
50% of N = N/2       (divide by 2)
66.67% of N = 2N/3
75% of N = 3N/4

Reverse trick — "x% of y = y% of x":
  37% of 200 = 200% of 37 = 2 × 37 = 74
  8% of 125 = 125% of 8 = 10 (since 125% = 5/4, and 8 × 5/4 = 10)
```

### 3.3 % Increase/Decrease — Multiplier Method
```
+10% → ×1.1   -10% → ×0.9
+20% → ×1.2   -20% → ×0.8
+25% → ×1.25  -25% → ×0.75
+50% → ×1.5   -50% → ×0.5
+100% → ×2    -100% → ×0

Successive % (multiply the multipliers):
+20% then −20% = 1.2 × 0.8 = 0.96 = −4%
+30% then −25% = 1.3 × 0.75 = 0.975 = −2.5%
+10% then +10% = 1.1 × 1.1 = 1.21 = +21%

Formula for successive a% and b% changes:
Net% = a + b + ab/100
  +20 and −20: 20+(−20)+(20×−20/100) = 0−4 = −4% ✓
  +10 and +10: 10+10+1 = 21% ✓
```

### 3.4 Reverse Percentage
```
To find original after a % change:
  After +20%, value is 120. Original? → 120/1.2 = 100
  After −25%, value is 75. Original? → 75/0.75 = 100
  After −15%, value is 340. Original? → 340/0.85 = 400
```

### 3.5 Finding What % of Each Other
```
A is what % of B? = (A/B) × 100

  48 is what % of 64? = (48/64)×100 = (3/4)×100 = 75%
  
  Shortcut: Use fraction tables!
  48/64 = 3/4 → 75% (from memory)
  
  35 is what % of 140? = (35/140)×100 = (1/4)×100 = 25%
```

## 🧠 SECTION 4: ADDITION & SUBTRACTION TRICKS

### 4.1 Rounding and Adjusting
```
Add 198: Add 200, subtract 2
  567 + 198 = 567 + 200 − 2 = 765

Add 297: Add 300, subtract 3
  843 + 297 = 843 + 300 − 3 = 1140

Subtract 98: Subtract 100, add 2
  567 − 98 = 567 − 100 + 2 = 469

Subtract 297: Subtract 300, add 3
  843 − 297 = 843 − 300 + 3 = 546
```

### 4.2 Grouping to Round Numbers
```
Add series by grouping pairs that sum to 100:
  23+77+45+55+12+88 = (23+77)+(45+55)+(12+88) = 100+100+100 = 300

Sum of consecutive integers:
  1+2+3+...+n = n(n+1)/2
  Sum of 1 to 100 = 100×101/2 = 5050
  Sum of 1 to 50 = 50×51/2 = 1275
```

### 4.3 Checking Addition (Casting Out Nines)
```
To verify: 4568 + 3427 = 7995?
  4568: 4+5+6+8=23→5
  3427: 3+4+2+7=16→7
  Sum: 5+7=12→3

  7995: 7+9+9+5=30→3 ✓ Correct!
```

## 🧠 SECTION 5: NUMBER SHORTCUTS

### 5.1 Squaring 2-digit Numbers — Quick Methods
```
Method: (a+b)² = a² + 2ab + b²
Where a = tens digit × 10, b = units digit

  47²: a=40, b=7
    = 1600 + 560 + 49
    = 2209 ✓

  83²: a=80, b=3
    = 6400 + 480 + 9
    = 6889 ✓

  Faster pattern for near-50:
  48² = (50−2)² = 2500−200+4 = 2304
  52² = (50+2)² = 2500+200+4 = 2704
  
  Near-100:
  97² = (100−3)² = 10000−600+9 = 9409
```

### 5.2 Products of Numbers with Same Sum
```
If a+b = c+d, then:
  ab and cd differ by: ab−cd = a² −(a−c)(a−d)/... 
  
  Better: (a)(a+k) vs (b)(b+k) where sum is same
  
  Useful identity: a² − b² = (a+b)(a−b)
  
  49 × 51 = (50−1)(50+1) = 50²−1 = 2499
  98 × 102 = (100−2)(100+2) = 10000−4 = 9996
  47 × 53 = (50−3)(50+3) = 2500−9 = 2491
  
  General: (n−k)(n+k) = n²−k²
```

### 5.3 Digital Root (Digit Sum) Table
```
A number ÷ 9: remainder = digital root

  N mod 9 = digital root of N (when root ≠ 9, else it's 0)
  
  Digital roots of squares: 1, 4, 7, 9, 7, 4, 1, 9, 9, 1, 4, 7...
  (Only 1, 4, 7, 9 possible as square digit roots)
  
  Quick test: is 54321 a perfect square?
  Digital root: 5+4+3+2+1=15→6 → 6 is NOT in {1,4,7,9} → NOT a perfect square ✓
```

## 🧠 SECTION 8: EXAM-SPECIFIC CALCULATION SHORTCUTS

### 8.1 SI & CI Quick Calculations
```
Simple Interest:
  SI = PRT/100
  For R=10%, T=3: SI = P×10×3/100 = 0.3P = 30% of P

Compound Interest shortcuts:
  At 10% for 2 years: A = P×1.21 → CI = 21% of P
  At 10% for 3 years: A = P×1.331 → CI = 33.1% of P
  At 20% for 2 years: A = P×1.44 → CI = 44% of P
  At 5% for 2 years: A = P×1.1025 → CI = 10.25% of P

Rule of 72:
  Years to double at r% = 72/r
  At 8%: doubles in 9 years
  At 12%: doubles in 6 years
  At 6%: doubles in 12 years
```

### 8.2 Speed-Distance-Time Fast Patterns
```
Conversion table (memorize!):
  18 km/h = 5 m/s      36 km/h = 10 m/s
  54 km/h = 15 m/s     72 km/h = 20 m/s
  90 km/h = 25 m/s     108 km/h = 30 m/s

Average speed (equal distance): 2ab/(a+b)
  60 and 40: 2×60×40/100 = 48 km/h
  50 and 50: 50 km/h (same speed, same average)
  
Pattern: Average speed < arithmetic mean for unequal speeds
  Quick calc: 2ab/(a+b) = 2/(1/a+1/b) [harmonic mean]
```

### 8.3 Work & Time Quick Patterns
```
Two workers: Combined rate = 1/A + 1/B = (A+B)/(AB)
Time together = AB/(A+B)

Quick pattern table:
  A=6, B=3: Together = 6×3/(6+3) = 2 days
  A=10, B=15: Together = 150/25 = 6 days
  A=12, B=12: Together = 6 days (half of either)
  A=n, B=n: Together = n/2 (always!)
  
If A is twice as fast as B:
  They work in ratio 2:1
  Together time = B-alone-time × 2/3 = A-alone-time/3 × ... 
  Simpler: If A=T/2 and B=T, together = T/3 × ... = AB/(A+B)
```

### 8.4 Profit/Loss Quick Patterns
```
After markup m% and discount d%:
  Net profit% = (100+m)(100−d)/100 − 100

  Mark 25%, discount 20%:
  = 125×80/100 − 100 = 100−100 = 0% (break even!)
  
  Mark 20%, discount 10%:
  = 120×90/100 − 100 = 108−100 = 8% profit

Same SP, profit a% on one, loss a% on other:
  Overall loss% = (a/10)² %
  At 10%: loss = 1%
  At 20%: loss = 4%
  At 25%: loss = 6.25%
```
