## 🔑 Reasoning Mental Models

### The Table Method
*Use for: Seating, Puzzles, Blood Relations*
- Draw a grid/table first
- Fill definite clues first
- Use elimination for rest

### The Venn Diagram Method
*Use for: Syllogism, Data Sufficiency*
- Draw circles for each statement
- Check if conclusion is always/sometimes/never true

### The Pattern Method
*Use for: Coding-Decoding, Series, Analogies*
- Find the rule: +N, ×N, alternating, positional
- Apply rule to unknown

### The Tree Method
*Use for: Blood Relations*
- Draw family tree top-down
- Male = Square, Female = Circle
- Marriage = Horizontal line

---

## ⚡ Reasoning Speed Tips

- **Syllogism**: Use Venn diagrams — never try to solve logically in head
- **Series**: Check differences first (+), then ratios (×), then squares
- **Coding**: Check +1, +2 shifts first; then reverse alphabet
- **Direction**: Always draw — never visualize in head
- **Seating**: Start with "definite" clues (A is at position 3)
