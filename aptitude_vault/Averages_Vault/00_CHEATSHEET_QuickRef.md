## 🔑 Core Formula
```
Average = Sum / n
Sum = Average × n          ← USE THIS MOST!
n = Sum / Average
```

---

## ⚡ Lightning Shortcuts

| Situation                 | Formula                                   |
| ------------------------- | ----------------------------------------- |
| Basic average             | Sum ÷ n                                   |
| **AP / Equal-gap series** | (First + Last) / 2                        |
| Odd count AP              | **Middle number**                         |
| 1 to n                    | (n+1)/2                                   |
| First n ODD numbers       | **n**                                     |
| First n EVEN numbers      | **n+1**                                   |
| Assumed mean              | A + avg(deviations)                       |
| Weighted average          | (n₁A₁ + n₂A₂)/(n₁+n₂)                     |
| **Replacement**           | Old value + n×(NewAvg−OldAvg)             |
| **New member joins**      | NewAvg + n×(NewAvg−OldAvg)                |
| Missing number            | (Avg×n) − Sum of rest                     |
| Wrong entry correction    | Correct avg = Old avg + (Correct−Wrong)/n |
| Time shift (years later)  | Avg + t                                   |
| **Equal distance speed**  | 2S₁S₂/(S₁+S₂)                             |
| Equal time speed          | (S₁+S₂)/2                                 |
| Any speed problem         | Total Distance / Total Time               |

## 🚨 Top 5 Exam Traps

1. **Speed equal distance** → Use Harmonic Mean, NOT simple average
2. **Average of averages** → Only valid if group sizes are equal
3. **Wrong number** → Change in avg = Error ÷ n  
4. **New member age** → Teacher older than students pulls avg UP more than you think
5. **Time shift** → Average age X years later = current avg + X

## 🧠 Memory Codes

```
SANe = Sum = Average × Number
ODD avg = n (same as count)
EVEN avg = n+1 (one more)
REPLACE = Old + n×ΔAvg
NEW MEMBER = NewAvg + n×ΔAvg
```
