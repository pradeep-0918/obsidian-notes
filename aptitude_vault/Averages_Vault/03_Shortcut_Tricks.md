## 🚀 TRICK 1 — Assumed Mean (The King of Shortcuts)

**When to use:** Numbers are large or inconvenient  
**Idea:** Assume a "fake average", find errors, correct it.

### Steps:
```
1. Pick any convenient number as Assumed Mean (A)
2. Find deviation of each number from A  
   (deviation = number − A)
3. Find average of deviations
4. Real Average = A + (Average of deviations)
```

### Example:
> Find average of: **198, 200, 202, 204, 196**

❌ Old way: 198+200+202+204+196 = 1000 ÷ 5 = 200

✅ Shortcut:
```
Assume A = 200
Deviations: -2, 0, +2, +4, -4
Sum of deviations = -2+0+2+4-4 = 0
Average deviation = 0 ÷ 5 = 0
Real Average = 200 + 0 = 200 ✅
```
**Saved massive addition work!**

## 🚀 TRICK 2 — Middle Number for Consecutive/Equal-gap Series

**When to use:** Numbers are in AP (equal gap between them)

> **Average of equally spaced numbers = Middle Number**

| Series | Average |
|--------|---------|
| 1, 2, 3, 4, 5 | **3** (middle) |
| 10, 20, 30, 40, 50 | **30** (middle) |
| 5, 10, 15, 20, 25 | **15** (middle) |

> 💡 **For even count:** Average = (middle two numbers) ÷ 2

| Series | Average |
|--------|---------|
| 2, 4, 6, 8 | (4+6)÷2 = **5** |

## 🚀 TRICK 3 — First & Last Rule

**For any AP (arithmetic progression):**

$$\text{Average} = \frac{\text{First} + \text{Last}}{2}$$

### Example:
> Average of 1 to 100?
```
= (1 + 100) ÷ 2 = 101 ÷ 2 = 50.5 ✅
(No need to add 1+2+3...+100 !)
```

> Average of 21, 23, 25, 27, 29?
```
= (21 + 29) ÷ 2 = 50 ÷ 2 = 25 ✅
```

## 🚀 TRICK 4 — Change in Average When One Number Changes

**Formula:**
```
Change in Sum = Change in number
Change in Average = Change in Sum ÷ n
```

### Example:
> Average of 5 numbers is 20.  
> One number increases by 10.  
> New average?

```
Change in Sum = +10
Change in Average = +10 ÷ 5 = +2
New Average = 20 + 2 = 22 ✅
```

## 🚀 TRICK 5 — Replacement Trick (⚡ 3-second formula)

**When a person leaves and another joins:**

$$\text{New person's value} = \text{Old person's value} + n \times (\text{New Avg} - \text{Old Avg})$$

### Example:
> Average of 5 people = 30 kg  
> One person (28 kg) is replaced  
> New average = 32 kg  
> Find new person's weight.

```
New person = 28 + 5 × (32 - 30)
           = 28 + 5 × 2
           = 28 + 10 = 38 kg ✅
```

## 🚀 TRICK 6 — New Member Joins Group

**When a new member joins:**

$$\text{New member's value} = \text{New Avg} + n \times (\text{New Avg} - \text{Old Avg})$$

*(where n = original count, before new member)*

### Example:
> Average of 4 people = 20 years  
> A new person joins, average becomes 21  
> Age of new person?

```
New person = 21 + 4 × (21 - 20)
           = 21 + 4 = 25 years ✅
```

## 🚀 TRICK 7 — Average of Consecutive Natural Numbers

| Type | Formula |
|------|---------|
| 1 to n | (n+1) ÷ 2 |
| 1 to n² | (n²+1) ÷ 2 |
| Consecutive even from 2 | (n+2) ÷ 2... just use First+Last÷2 |
| Consecutive odd | Middle number |

> **Average of 1 to 50 = (50+1)/2 = 25.5**  
> **Average of 1 to 99 = (99+1)/2 = 50**
