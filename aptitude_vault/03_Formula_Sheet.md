## 🕉️ VEDIC MATHS QUICK REFERENCE

### Squaring Shortcuts
| Number Type | Method | Example |
|---|---|---|
| Ends in 5 | n×(n+1) then \|25 | 75² = 7×8\|25 = **5625** |
| Near 100 (below) | (n−d)\|d² | 97² = 94\|09 = **9409** |
| Near 100 (above) | (n+d)\|d² | 103² = 106\|09 = **10609** |
| (a−b)² | a²−2ab+b² | 48² = 2500−200+4 = **2304** |
| (a+b)² | a²+2ab+b² | 52² = 2500+200+4 = **2704** |

### Multiplication Shortcuts
| Multiplier | Method | Example |
|---|---|---|
| ×11 | A\|(A+B)\|B | 47×11 = 4\|11\|7 = **517** |
| ×9 | n×10−n | 47×9 = 470−47 = **423** |
| ×99 | (n−1)\|(100−n) | 47×99 = 46\|53 = **4653** |
| ×999 | (n−1)\|(1000−n) | 63×999 = 62\|937 = **62937** |
| ×25 | n×100÷4 | 48×25 = 4800÷4 = **1200** |
| ×125 | n×1000÷8 | 24×125 = 24000÷8 = **3000** |
| ×5 | n×10÷2 | 47×5 = 470÷2 = **235** |

### Near-Base Multiplication (Nikhilam)
```
Both near 100: deficiency method
  97 × 96: def=3,4 → L=97−4=93 | R=3×4=12 → 9312
  103×105: exc=3,5 → L=103+5=108| R=3×5=15 → 10815
```

### Cube Root Instant Method
```
Units digit clues: 1→1, 2→8, 3→7, 4→4, 5→5, 6→6, 7→3, 8→2, 9→9
Remove last 3 digits → find cube containing left part
∛17576: units=6→6, left=17, 2<17<27 → 2 → answer: **26**
```

### Digit Sum Verification
```
Multiply digit sums of inputs → compare with digit sum of answer
47×83=3901: DS(47)=2, DS(83)=2, 2×2=4; DS(3901)=4 ✓
```

### Percentage Mental Math
```
10%: move decimal left  →  10% of 340 = 34
5%:  half of 10%        →  5% of 340 = 17
15%: 10% + 5%          →  15% of 340 = 51
25%: divide by 4        →  25% of 340 = 85
33%: divide by 3        →  33% of 342 = 114
75%: ÷4 then ×3        →  75% of 340 = 255

"x% of y = y% of x" → 37% of 200 = 200% of 37 = 74
```
