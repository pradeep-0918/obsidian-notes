# 🔗 Analogies

## 🧠 Concept

An analogy defines a **relationship** between two things, and you must find the same relationship in a new pair.

**A : B :: C : ?**
"A is to B as C is to ?"

---

## 📌 Relationship Categories

### Category 1: Part → Whole
> Wheel : Car :: Petal : Flower
> Leg : Table :: Page : Book

### Category 2: Worker → Tool / Tool → Worker
> Carpenter : Saw :: Surgeon : Scalpel
> Pen : Writer :: Brush : Painter

### Category 3: Animal → Young one
> Cat : Kitten :: Dog : Puppy
> Cow : Calf :: Horse : Foal

### Category 4: Animal → Home / Sound
> Bird : Nest :: Fish : River
> Dog : Bark :: Cat : Meow

### Category 5: Worker → Workplace
> Teacher : Classroom :: Doctor : Hospital
> Chef : Kitchen :: Judge : Courtroom

### Category 6: Action → Tool
> Write : Pen :: Cut : Scissors

### Category 7: Degree/Intensity
> Warm : Hot :: Cool : Cold
> Smile : Laugh :: Dislike : Hate

### Category 8: Cause → Effect
> Practice : Perfection :: Rain : Flood

### Category 9: Synonyms / Antonyms
> Big : Large :: Fast : Quick (synonyms)
> Big : Small :: Fast : Slow (antonyms)

### Category 10: Number/Science Relationships
> Square : 4 sides :: Triangle : 3 sides
> Biologist : Life :: Geologist : Earth

---

## 🛠️ Solving Strategy

1. **State the relationship clearly in words:**
   > "Pen is a tool used for writing"
2. **Apply same relationship to C:**
   > "Brush is a tool used for ?"
3. **Find the matching answer:**
   > **Painting**

### The One-Sentence Test
Always frame: "A **[relationship]** B, just as C **[same relationship]** D"
> "A pen is used for writing, just as a brush is used for **painting**"

---

## 💡 Shortcuts

- **If two relationships seem possible:** Pick the more SPECIFIC/PRECISE one
- **Word order matters:** A:B and B:A are different analogies
  > "Cat : Meow" (animal to sound) ≠ "Meow : Cat" (sound to animal)
- **Eliminate clearly wrong options first**
- **Look for same grammatical form:** noun:noun, verb:verb

## 🧩 Practice Questions

### Easy
1. Big : Small :: Fast : ? → **Slow** (antonyms)
2. Cat : Kitten :: Dog : ? → **Puppy** (young ones)
3. Pen : Write :: Knife : ? → **Cut** (tool:action)
4. Teacher : Classroom :: Chef : ? → **Kitchen** (worker:workplace)
5. Bird : Fly :: Fish : ? → **Swim** (animal:movement)

### Medium
6. Heavy : Light :: Big : ? → **Small** (antonyms)
7. Bird : Nest :: Spider : ? → **Web** (animal:home)
8. Doctor : Medicine :: Chef : ? → **Food/Recipe** (worker:output)
9. Rain : Cloud :: Snow : ? → **Ice/Cold air** (effect:cause)
10. Eye : Vision :: Nose : ? → **Smell** (organ:sense)

### Difficult
11. Big : Small :: Bright : ? → **Dim** (antonyms, degree)
12. Cat : Feline :: Dog : ? → **Canine** (animal:scientific family)
13. Pen : Ink :: Gun : ? → **Bullet** (tool:ammunition it uses)
14. Book : Library :: Painting : ? → **Gallery/Museum**
15. Sun : Solar :: Moon : ? → **Lunar** (celestial body:adjective form)
