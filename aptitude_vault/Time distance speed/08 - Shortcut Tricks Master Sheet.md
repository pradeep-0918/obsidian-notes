## ⚡ CATEGORY 2: Speed-Time Ratio Tricks

### Trick 1: Inverse Proportion (Fixed Distance)
> If speed increases by fraction **a/b**, time DECREASES by **a/(a+b)**

**Example:** Speed increases by 1/4 (i.e., 25%)
Time decreases by 1/(4+1) = **1/5** (i.e., 20%)

---

### Trick 2: Speed Ratio → Time Ratio
> Same distance: Speed₁ : Speed₂ = Time₂ : Time₁ (reversed!)

**Example:** If A is 3× faster than B → A takes 1/3 the time.

---

### Trick 3: The "Late/Early" Trick
> A person arrives T₁ minutes late at speed S₁, T₂ minutes early at speed S₂.
> Correct time to reach = ?

$$\text{Time} = \frac{S_1 \times T_2 + S_2 \times T_1}{S_2 - S_1}$$

$$\text{Distance} = \frac{S_1 \times S_2 \times (T_1 + T_2)}{S_2 - S_1}$$

**Example:** 5 km/h → 10 min late. 6 km/h → 12 min early.

Distance = (5×6×22)/(6-5) = 5×6×22 = **660 m**

---

### Trick 4: Missing Train Shortcut
> If you just missed train by T minutes, and next train is at distance D:
> Required speed = D / T

## ⚡ CATEGORY 3: Average Speed Tricks

### Trick 5: Harmonic Mean Shortcut
> Two speeds, equal distance: Avg = 2ab/(a+b)

Quick estimate: **Avg is always closer to the smaller speed!**

### Trick 6: One Speed Double Another
> If S₂ = 2×S₁ (same distance):
> Avg = **4/3 × S₁** = **2/3 × S₂**

Example: 20 km/h and 40 km/h → Avg = 2/3 × 40 = **26.67 km/h**

### Trick 7: 3 Speeds Equal Distance
$$\text{Avg} = \frac{3abc}{ab+bc+ca}$$

## ⚡ CATEGORY 4: Train Tricks

### Trick 8: Train + Pole Time = Train Length Formula
> Length = Speed × Time(crossing pole)

### Trick 9: Find Length Using Two Times
> Length = Platform × t₁ / (t₂ - t₁)
> (t₁ = time for pole, t₂ = time for platform)

### Trick 10: Two Trains — Passing Each Other
- Opposite: use Sum of speeds
- Same: use Difference of speeds
- Distance = Sum of their lengths (ALWAYS)

## ⚡ CATEGORY 5: Boats & Streams Tricks

### Trick 11: Boat and Stream Formula
$$B = \frac{D+U}{2}, \quad S = \frac{D-U}{2}$$

### Trick 12: Time Ratio Formula
$$\frac{T_{up}}{T_{down}} = \frac{D_{speed}}{U_{speed}}$$

### Trick 13: Man Overboard
> Stream speed = Total distance floated / Total time elapsed
> (Ignore boat speed completely!)

## ⚡ CATEGORY 6: Relative Speed Tricks

### Trick 14: Meeting Ratio Split
> Two people from ends A and B: they split distance in ratio of speeds
> A covers: (S_A / (S_A + S_B)) × Total Distance

### Trick 15: Circular Track Meetings
| Direction | Time to meet |
|-----------|-------------|
| Same | Track / \|S₁-S₂\| |
| Opposite | Track / (S₁+S₂) |

### Trick 16: How Many Times They Meet
> In time T, total meetings = T × relative_speed / track_length

## ⚡ CATEGORY 7: Percentage/Fraction Shortcuts

### Trick 17: Speed % Change → Time % Change Table

| Speed Change | Time Change |
|---|---|
| +10% (×11/10) | -1/11 ≈ -9.09% |
| +20% (×6/5) | -1/6 ≈ -16.67% |
| +25% (×5/4) | -1/5 = -20% |
| +33.3% (×4/3) | -1/4 = -25% |
| +50% (×3/2) | -1/3 ≈ -33.33% |
| +100% (×2) | -1/2 = -50% |
| -20% (×4/5) | +1/4 = +25% |
| -25% (×3/4) | +1/3 ≈ +33.33% |
| -33.3% (×2/3) | +1/2 = +50% |

> 📌 **Rule:** If speed multiplied by m/n → time multiplied by n/m

## ⚡ CATEGORY 8: The Distance Splitting Trick

When Person A starts from X and Person B from Y toward each other:

**After first meeting:**
> Each person then travels to the other end and comes back.
> The **second meeting** from A's start = 3× the distance covered by A at first meeting!

Example: If A covers 60 km at first meeting in a 100 km path:
- First meeting: A at 60 km mark
- Second meeting: A at 3×60 - 100 = **80 km mark** (if >100, subtract 100)

## ⚡ CATEGORY 9: Clock Problems (Bonus!)

### Trick 18: Minute Hand vs Hour Hand
- Minute hand: 6° per minute
- Hour hand: 0.5° per minute
- Relative speed: 5.5° per minute

**Time for hands to be at angle θ:**
$$t = \frac{\theta}{5.5} \text{ minutes}$$

**Overlap (0°):** Every 65.45 min = 720/11 minutes

## 🚨 Top 10 Mistakes to NEVER Make

| ❌ Common Mistake | ✅ Correct Approach |
|-----------------|-------------------|
| Average speed = (S1+S2)/2 always | Only when time is equal |
| Forgetting to add lengths in train problems | Always add lengths |
| Using km/h when problem gives m/s | Convert units first |
| Not converting minutes to hours | 30 min = 0.5 hr, not 0.3 |
| Subtracting speeds when going opposite | Opposite = ADD speeds |
| Adding speeds when going same way | Same = SUBTRACT speeds |
| Forgetting boat speed ≠ downstream speed | Downstream = B + S |
| Using simple distance = S×T when units mismatch | Always check units |
| Missing "head start" in chasing problems | Apply head start first |
| Solving circular meeting as linear | Use track length as distance |

## 🧠 The 5-Second Scan Method (Exam Strategy)

Before solving ANY TSD problem, ask:

1. **Units?** → Are all units the same? Convert if needed.
2. **Type?** → Basic / Average / Relative / Train / Boat?
3. **Direction?** → Same or opposite?
4. **Lengths?** → Any objects with length? (trains, platforms)
5. **Trick?** → Can I use a shortcut from this sheet?
