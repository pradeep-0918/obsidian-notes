## 🔺 THE MAGIC TRIANGLE

```
        D
      -----
      S × T
```
- **D** = S × T
- **S** = D ÷ T
- **T** = D ÷ S

---

**km/h → m/s:** × 5/18
**m/s → km/h:** × 18/5

## 📐 AVERAGE SPEED

| Condition | Formula |
|-----------|---------|
| Same **distance** at S₁ and S₂ | 2S₁S₂/(S₁+S₂) |
| Same **time** at S₁ and S₂ | (S₁+S₂)/2 |
| General | Total Distance / Total Time |
| S₂ = 2×S₁ (equal dist) | 4S₁/3 or 2S₂/3 |

---

## 🚦 RELATIVE SPEED

| Scenario            | Speed             |
| ------------------- | ----------------- |
| Opposite direction  | S₁ + S₂           |
| Same direction      | \|S₁ - S₂\|       |
| Circular — Same     | Track / \|S₁-S₂\| |
| Circular — Opposite | Track / (S₁+S₂)   |

**Meeting point split:** Distances in ratio of speeds

---

## 🚂 TRAINS

| Crossing          | Distance         | Speed       |
| ----------------- | ---------------- | ----------- |
| Pole/Person       | Train length     | Train       |
| Platform/Bridge   | Train + Platform | Train       |
| Trains (opposite) | L₁ + L₂          | S₁ + S₂     |
| Trains (same)     | L₁ + L₂          | \|S₁ - S₂\| |

**Find length from 2 times:**
$$L_{train} = L_{platform} \times \frac{t_{pole}}{t_{platform} - t_{pole}}$$

## 🚣 BOATS & STREAMS

| | Formula |
|--|---------|
| Downstream | B + S |
| Upstream | B - S |
| Boat speed | (D + U) / 2 |
| Stream speed | (D - U) / 2 |
| Time ratio | T_up/T_down = D_speed/U_speed |

---

## ⏰ LATE/EARLY TRICK

> Late by T₁ at speed S₁, Early by T₂ at speed S₂:

$$D = \frac{S_1 \times S_2 \times (T_1 + T_2)}{S_2 - S_1}$$

---

## 📊 SPEED CHANGE → TIME CHANGE

| Speed | Time |
|-------|------|
| ×2 | ×1/2 |
| ×3/2 (+50%) | ×2/3 (-33%) |
| ×5/4 (+25%) | ×4/5 (-20%) |
| ×4/3 (+33%) | ×3/4 (-25%) |
| ×4/5 (-20%) | ×5/4 (+25%) |

**Rule:** Multiply speed by m/n → multiply time by n/m

---

## 🏁 AFTER-CROSSING TRICK

> If after crossing, A takes time t₁ and B takes time t₂ to finish:
$$\frac{Speed_A}{Speed_B} = \sqrt{\frac{t_2}{t_1}}$$

## 🧭 5-SECOND PROBLEM SCANNER

```
1. Units same? → Convert if not
2. Type? Basic / Relative / Train / Boat / Average
3. Direction? Same (+subtract) | Opposite (+add)
4. Has length? Always add lengths for trains!
5. Shortcut available? Check this sheet!
```

---

## 🚨 NEVER MAKE THESE MISTAKES

1. ❌ Average = (S1+S2)/2 when distances are equal → ✅ Harmonic mean
2. ❌ Forgetting to add train + platform lengths → ✅ Always add
3. ❌ 30 min = 0.3 hours → ✅ 30 min = 0.5 hours
4. ❌ Opposite direction = subtract speeds → ✅ Opposite = ADD
5. ❌ Downstream = Boat speed → ✅ Downstream = B + S
