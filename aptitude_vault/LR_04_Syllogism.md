## рҹ§  Concept

Syllogism tests **logical deduction** вҖ” given statements, determine which conclusions MUST be true.

**Key principle:** You must accept statements as TRUE (even if they seem false in real life) and check if conclusions NECESSARILY follow.

> "All cats are dogs. All dogs are fish." вҶ’ Even though absurd, "All cats are fish" is TRUE logically!

## рҹ”Ҙ Conclusion Rules (CRITICAL)

### Rule 1: All + All вҶ’ All
> All A are B. All B are C. вҶ’ **All A are C.** вң“

### Rule 2: All + No вҶ’ No
> All A are B. No B is C. вҶ’ **No A is C.** вң“

### Rule 3: Some + All вҶ’ Some
> Some A are B. All B are C. вҶ’ **Some A are C.** вң“

### Rule 4: Some + No вҶ’ Some not
> Some A are B. No B is C. вҶ’ **Some A are not C.** вң“

### Rule 5: All + Some вҶ’ No definite conclusion
> All A are B. Some B are C. вҶ’ **Cannot conclude anything definite about A and C.**

### Rule 6: Some + Some вҶ’ No definite conclusion
> Some A are B. Some B are C. вҶ’ **Cannot conclude A and C are related.**

### Rule 7: No + No вҶ’ No definite conclusion
> No A is B. No B is C. вҶ’ **Cannot conclude about A and C.**

## рҹ’Ў Quick Validity Checklist

For a conclusion to be valid:
1. Draw Venn diagrams for all statements
2. Check: Does the conclusion ALWAYS hold for ALL possible valid diagrams?
3. If even ONE valid diagram violates the conclusion вҶ’ conclusion is FALSE
4. "Possibility" conclusions (Some A may be C) вҶ’ True if not contradicted

## рҹ”Ҙ Types of Conclusions

### Type 1: Direct Conclusion
> Statements: All books are pens. Some pens are pencils.
> Conclusion: Some books are pencils.
> **Answer: Invalid** (Rule 5: All+Some вҶ’ No conclusion)

### Type 2: Complementary Pair
> Conclusions: I. Some A are B. II. No A is B.
> If neither is definitively true, check if they form a complementary pair (one must be true):
> Some A are B AND No A are B are complementary вҶ’ **One must follow** (called "Either-Or")

### Type 3: Reverse Conclusion
> Statement: All A are B. Conclusion: All B are A.
> **Invalid!** Just because all A are in B doesn't mean all B are A.

### Type 4: Combination
> All A are B. All A are C. вҶ’ Some B are C? 
> **Valid!** (Since A is in both, B and C overlap through A at minimum)

## вҡ пёҸ Common Mistakes

1. **Real-world interference**
   - "All men are dogs" seems wrong, but in syllogism, accept as true.

2. **Reversing "All"**
   - "All A are B" does NOT mean "All B are A"

3. **Confusing "Some" and "All"**
   - "Some A are B" means at least one, possibly all

4. **Not checking BOTH conclusions**
   - In either-or questions, both might be wrong, or exactly one right

5. **Forgetting indirect conclusions**
   - Chain through multiple statements to reach conclusion

## рҹ§© Practice Questions

### Medium
1. All books are pens. Some pens are pencils.
   Conclusions: I. Some books are pencils. II. Some pencils are books.
   > I: Books-pens-pencils: Some(not All) pens are pencils, but books are all-pens; Can't guarantee books touch pencils вҶ’ **Invalid**
   > II: Since some pens are pencils and all books are pens: books are a subset of pens. Some pencils overlap with pens but not necessarily books вҶ’ **Invalid**
   > **Neither follows**

2. All cats are dogs. All dogs are animals.
   Conclusions: I. All cats are animals. II. All animals are dogs.
   > I: All catsвҶ’dogsвҶ’animals, so all cats are animals вҶ’ **Valid (follows)**
   > II: We only know dogs are animals, but animals may not all be dogs вҶ’ **Invalid**

3. Some A are B. All B are C. Some C are D.
   Conclusions: I. Some A are C. II. Some D are A.
   > I: Some A are B, All B are C вҶ’ Some A are C вҶ’ **Valid**
   > II: Some C are D, Some A are C вҶ’ A and D may or may not overlap вҶ’ **Invalid**

4. All roses are flowers. Some flowers are red.
   Conclusions: I. Some roses are red. II. Some red are flowers.
   > I: All roses are flowers, some flowers are red вҖ” roses might be in the non-red part вҶ’ **Invalid**
   > II: Some flowers are red вҶ’ reversing "some" вҶ’ Some red are flowers вҶ’ **Valid**

5. No pen is marker. Some markers are blue.
   Conclusions: I. No pen is blue. II. Some blue are not pens.
   > I: Pens and markers are separate. Some markers are blue. But blue things could still include pens (pens and blue are not restricted) вҶ’ **Invalid**
   > II: Since some markers are blue, and no marker is a pen, those blue markers are not pens вҶ’ **Valid**

6. All students are educated. Some educated are teachers.
   I. Some students are teachers. II. All educated are students.
   > I: Students are educated; some educated are teachers; teachers could be in the non-student educated part вҶ’ **Invalid**
   > II: All students are educated, but educated doesn't have to be all students вҶ’ **Invalid**
   > **Neither follows**

### Difficult
1. Some A are B. All B are C. Some C are D. No D is E.
   Conclusions: I. Some A are D. II. Some C are not E.
   > I: Some AвҶ’BвҶ’C; Some C are D. A and D: A is a subset of C via B, but only "some C are D" вҖ” might not include the A-part вҶ’ **Invalid (not definite)**
   > II: Some C are D, No D is E вҶ’ Some C (those that are D) are not E вҶ’ **Valid**

2. All X are Y. Some Y are Z. No Z is W. Conclusions: I. No X is W. II. Some Y are not W.
   > I: All X are Y; Some Y are Z; No Z is W. XвҠҶY. Some Y are Z which are not W. But X might be in the Y-not-Z portion вҶ’ **Invalid (cannot conclude No X is W)**
   > II: Some Y are Z, No Z is W вҶ’ those Z-type Y are not W вҶ’ Some Y are not W вҶ’ **Valid**

3. No X is Y. Some Y are Z. All Z are W.
   I. No X is Z. II. Some W are Y.
   > I: X and Y are separate. Z overlaps with Y. But X and Z: Not directly related вҖ” X could overlap with Z if Z extends beyond Y... but wait "Some Y are Z" doesn't mean all Z are Y. X and Z might still intersect. вҶ’ **Invalid (uncertain)**
   > II: Some Y are Z, All Z are W вҶ’ Some Y are ZвҶ’W, so Some Y are W вҶ’ Some W are Y (reverse of some) вҶ’ **Valid**
