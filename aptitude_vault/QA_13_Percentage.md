## 🧠 Concept

"Per cent" = per hundred. x% of N = x/100 × N.

**Core intuition:** 20% of 500 = 100 (one-fifth of 500)
> Learn to convert % ↔ fractions mentally!

---

## 📌 Key Conversions (Memorize!)

| Fraction | % | Fraction | % |
|---|---|---|---|
| 1/2 | 50% | 1/3 | 33.33% |
| 1/4 | 25% | 2/3 | 66.67% |
| 1/5 | 20% | 1/6 | 16.67% |
| 1/8 | 12.5% | 1/7 | 14.28% |
| 1/9 | 11.11% | 1/11 | 9.09% |

## 📌 Key Formulas

### Basic
$$x\% \text{ of } N = \frac{x \times N}{100}$$

### Percentage Change
$$\% \text{ Change} = \frac{\text{New} - \text{Old}}{\text{Old}} \times 100$$

### Increase/Decrease
- Increased by x%: New = Old × (1 + x/100)
- Decreased by x%: New = Old × (1 − x/100)

### Reverse Percentage (Find original)
$$\text{Original} = \frac{\text{New}}{1 + x/100}$$

### Successive Changes
Two changes a% and b%: Net = a + b + ab/100

## 🔥 Mental Math Tricks

### Trick 1: 10% + build
- 10% of 350 = 35
- 15% = 10% + 5% = 35 + 17.5 = 52.5
- 25% = 10% + 10% + 5% = 35+35+17.5 = 87.5

### Trick 2: Use fractions
- 25% of 80 = 1/4 × 80 = **20** (instant!)
- 33.33% of 600 = 1/3 × 600 = **200**
- 12.5% of 160 = 1/8 × 160 = **20**

### Trick 3: Increase/decrease by same % ≠ same result
> +20% then −20% → Net = 1.2 × 0.8 = 0.96 = **−4%** (not 0!)

### Trick 4: Price and quantity inverse for constant total spend
> Price up 25% → Quantity must drop by 20% to keep spending same
> (Because 1/1.25 = 0.8, i.e., 20% drop)

## 🧩 Practice Questions

### Easy
1. 20% of 150? → **30**
2. 80 increased by 25%? → 80×1.25 = **100**
3. 200 decreased by 10%? → 200×0.9 = **180**
4. 15% of ₹500? → **₹75**
5. Price ₹100 increased by 20%. New? → **₹120**

### Medium
6. +20% then −20%. Start = 100. End? → 100×1.2×0.8 = **96** (4% loss)
7. Price +25% to ₹125. Original? → 125/1.25 = **₹100**
8. +30% then −20%. Final if start = 100? → 100×1.3×0.8 = **104**
9. A is 20% more than B. B = 50. A? → 50×1.2 = **60**
10. Price −25% to ₹150. Original? → 150/0.75 = **₹200**

### Hard
11. After +20% then −20%, result = 96. Original?
    > 96 = x × 0.96; x = **100**
12. +25% then −20%. Result = ₹1200. What if +30% instead?
    > x×1.25×0.8 = 1200; x = 1200; new = 1200×1.3×0.8 = **₹1248**
13. Price +10% twice. Original = ₹100. Final?
    > 100×1.1×1.1 = **₹121** (not ₹120!)
14. +15% then +10%. Original = 200. Final?
    > 200×1.15×1.1 = 200×1.265 = **₹253**
15. Salary −10% then +15%. Final if original = ₹1000?
    > 1000×0.9×1.15 = **₹1035**
