## 🏛️ 4 Key Pillars of Linux Mastery

```
1. 🗂️  FILE SYSTEM  →  Understand directories, permissions, links
2. ⚙️  PROCESSES    →  Control what runs, when, and how
3. 🌐  NETWORKING   →  SSH, SCP, remote access
4. 🤖  AUTOMATION   →  Shell scripts + cron jobs
```

## ⚡ Quick Revision: Most Used Commands

```bash
# Navigation
pwd | ls | cd | mkdir | touch | rm

# Info
whoami | uname -a | hostname | id

# Files
cat | head | tail | grep | find | awk | sort | wc

# Processes  
ps aux | top | kill | nohup

# Disk
df -h | du -sh | zip | tar

# Network
ssh | scp | wget | apt-get

# Permissions
chmod | chown | ls -l
```
