## 🧠 What is Scala?

Scala = **Sca**lable **La**nguage.
Created by **Martin Odersky** in 2004 at EPFL, Switzerland.
It runs on the **JVM (Java Virtual Machine)** — meaning it can use all Java libraries!

> **For a 6th grader:** Imagine you have a toy box (JVM) that already has millions of LEGO pieces (Java libraries). Scala lets you build NEW, even cooler things with those same pieces, but faster and smarter.
